## About
Welcome to the `DocIDVAI` repository. 

It provides the official Swift Package for the `DocIDVAI` iOS SDK in charge of capture identity documents using the device's camera, performing several security checks and offering various other features.

⚠️ This SDK is not intended to work lonely but must be used through `DocIDV` sdk. Please refer https://github.com/idnow/docidv-sdk-ios for more details about external use.
